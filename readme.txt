== Description ==

The Advanced Wordpress Theme Editor plugin will allow you to edit your theme directly in your blog admin section but with a better experience since it will format and highlight your source code with nice colours.

The plugin has been tested with 2.8 up to 2.9.1 and is based on the Edit Area script that supports at least PHP, CSS, JS and many more file types. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload plug-in folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
